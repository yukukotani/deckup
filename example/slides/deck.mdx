# Markdown-first slides

Write prose in Markdown, keep the deck in one selected `.mdx` file, and split pages with post-frontmatter dividers.

  <ul className="mx-auto grid max-w-3xl gap-4 text-left md:grid-cols-3">
    <li className="rounded-2xl border border-purple-300/40 bg-purple-500/10 p-4 text-purple-950 dark:text-purple-100">
      Frontmatter sets the deck title.
    </li>
    <li className="rounded-2xl border border-purple-300/40 bg-purple-500/10 p-4 text-purple-950 dark:text-purple-100">
      `---` after frontmatter starts a new page.
    </li>

# Compose with JSX

Use Markdown for text-heavy sections, then add components or HTML-like JSX when a slide needs structure.

  <div className="mx-auto grid max-w-2xl gap-3 text-left">

    <div className="rounded-2xl border border-slate-300/50 p-4 dark:border-slate-700">
      Build: <code>slida build slides/deck.mdx</code>
